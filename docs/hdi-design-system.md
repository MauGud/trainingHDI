# Base De Diseño HDI Seguros México

Esta guía define el criterio visual base del proyecto. La interfaz debe sentirse institucional, corporativa, confiable y alineada a HDI Seguros México.

## Principios

- Consistencia visual sobre creatividad.
- Claridad operativa sobre estética.
- Fidelidad a marca sobre diferenciación.
- Sin gradientes, sombras fuertes, ilustraciones decorativas ni animaciones llamativas.
- Sin colores fuera de la paleta definida.

## Paleta

- Verde principal: `#006341`
- Verde hover/énfasis: `#004B32`
- Rojo crítico: `#E30613`
- Blanco: `#FFFFFF`
- Fondo suave: `#F5F5F5`
- Borde: `#E5E7EB`
- Texto secundario: `#6B7280`
- Texto principal: `#1A1A1A`
- Advertencia moderada: `#F59E0B`

## Componentes Obligatorios

Toda pantalla debe construirse con:

- `HDIHeader`
- `HDIFooter`
- `PageContainer`
- `SectionHeader`
- `PrimaryButton`
- `SecondaryButton`
- `Card`
- `Input`
- `Select`
- `Table`
- `Alert`

## Reglas Para Nuevas Pantallas

1. Usar header institucional y footer institucional.
2. Respetar ancho máximo de `1200px`.
3. Usar fondo blanco o gris claro.
4. Mantener jerarquía tipográfica simple y sobria.
5. Usar botones y formularios consistentes.
6. Reservar el rojo para acciones críticas o alertas.
7. Evitar cualquier estilo que parezca startup SaaS, fintech o landing de marketing agresiva.
